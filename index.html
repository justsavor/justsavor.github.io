First Post（如何搭建博客）

第一篇试用文就瞎写写试试博客效果
尝试说一下搭建博客的过程

事前准备

第一步：选择搭建博客的工具

简单的就是使用十几年前那种简易博客网站，新浪博客之类的
新一点的其实像简书，小红书之类的我觉得都可以算是博客的一种
我比较闲所以选择了相对Geek一点的方式
这类方式大体有2种

  1. 云服务器 + 域名 + Wordpress
    购买云服务器(阿里云之类的)和域名（万网，GoDaddy之类的），然后使用WordPress进行搭建，我大概了解了下这种方式，看着比较复杂，还有人自己写了前端的，对于完全没接触过代码什么的人来说难度有点高。而且很多教程上来就会先说购买服务器和域名的事情，虽然不是很贵，但是这种事，大家懂的，很可能半途而废，没必要马上花钱。不过如果愿意折腾花时间学，而且搭建的博客可能会有其他用途，图片量大内容特别多的话，还是比较推荐这种方式
    教程的话百度找一找也很多，知乎上也能搜到相应的回答
  2. GithubPage + Hexo
    这种方式是我觉得相对上一种要简单一些的方法，可以暂时不管服务器和域名的事情，先自己试着搭建出一个本地的博客看看效果，然后再视情况要不要加上域名什么的让别人也能访问。不过多少也需要用到一些命令行之类的感觉Geek的东西= =毕竟Github是程序员的大型社交网站，专业的程序员我觉得Github本身都是一种更加专业性的博客了，然后GithubPage是Github推出给用户使用的展示型的网站吧，用于程序员们展示自己的项目，互相探讨之类的。 Hexo就是一个博客框架了，专门用来搭建博客的。

第二步：找几个合适的教程，多百度谷歌

这种事情么现实中如果没有认识的人做过的话只能靠搜索了，遇到的问题基本都能找到解决办法。说到这个，感觉真的，如果用Mac做这类事情真的会方便很多，总觉得Windows会碰到奇奇怪怪的问题，码字之类的也是Mac用起来更舒服一点。
附上我主要看的教程《Windows下使用Hexo+GithubPage搭建博客》

自己的搭建过程

第一步：安装各种东西

  1. Node.js
    这是要用Hexo必备的前提应用程序，干吗用的不用管了去官网下载个安装包装好就是了

  2. Git
    Git和Github不是一个东西，Git是用来做版本控制的工具，单独的东西，Github是一个平台，让用户使用Git来做版本控制的托管平台。大概像是魔兽争霸和对战平台这种感觉。。。？总之是单独的东西，我一开始安装的时候只装了一个Github的桌面程序，以为就装好了，发现并不是这样。
    Git的下载地址：https://git-scm.com/
    进去之后点击左下角的Download选择相应的系统平台即可
    Github桌面程序点击右下角的Windows GUIs，进去之后选择Github Desktop
    Git的安装过程中的选项，参考我上面附上的教程链接中的选项，我就不重新贴图了

  3. Github
    刚才提到了它是一个版本控制的托管平台，一般称之为程序员同性交友网站。在搭建博客中主要用来托管我们的博客，以达到类似于在网站发布的功能。
    首先没有Github账号的人要注册一个账号，我做毕设的时候有用过Github所以跳过了这个步骤。注册账号的过程和别的网站没什么区别。进官网一步步注册就行了。
    注册完之后要添加SSH Key，有了这个Key才能让Github识别你的电脑，允许电脑提交代码，同步博客。
    这里那个教程中写的不是很清楚，我说详细一点
    首先登录自己的Github账号，然后点击右上角自己的头像，选择Settings

    然后选择SSH and GPG Keys
    我这边已经有Key了，所以显示了我的Key，点击下面那个generating SSH Keys的链接

    再点击Generating a new SSH key and adding it to the ssh-agent

    就会看到这个生成Key的过程，很详细

    其中第一步里Git Bash是安装完Git后就会有的东西，开始菜单里能找到

    打开之后就是上面那个黑底的看起来像是命令行一样的东西，照着输就行了
    之后就按照教程里找到id_rsa.pub这个文件，用记事本打开，复制里面的Key到Github的网站上添加就行了

  4. Hexo
    安装Hexo的时候就按照教程或者Hexo官网的步骤一步步做就行了，没碰到什么问题
    ####配置GithubPage
    登录Github，然后点击New Repository，输入名字之类的，就按照教程做，但是有一点

    这个得勾上，不然后面的步骤做不了，除非自己手动加一个文件进这个Repository
    然后要启用GithubPage，和教程里的图片有点不一样

    往下滑看到GitHub Pages

    下面选择Master Branch，Theme随便选一个就行了，然后保存，创建就可以了。这之后进入http://(你的名字).github.io就可以看到一个个人网站一样的东西了

将Hexo项目托管到Github上

这个相当于将安装Hexo创建的项目放到Github上，将Hexo创建的博客和github.io的链接绑定起来，这样以后进入github.io就可以直接看到这个博客。按照教程一步步做就行了。
